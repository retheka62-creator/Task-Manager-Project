 ## Task Management Web Application
Task Management Web Application is a responsive web application built using Python Flask, HTML, CSS, JavaScript, and SQLite/MySQL. It helps users manage daily tasks efficiently with secure authentication, task tracking, search, filtering, and dashboard statistics.

## Project Overview:
This project allows users to:
Register and login securely
Create, update, delete, and manage tasks
Set task priority levels
Add due dates
Mark tasks as completed
Search and filter tasks
Track task progress through dashboard statistics
Switch between dark mode and light mode
Use the application on desktop and mobile devices
